Software Requirements
=====================
First you will need to install a few libraries to compile the source

Libraries:
* Mono SDK (>= 1.9.1)
* Soup-sharp ( optionel but i like it :) )

Compiling Source
================

    cd src && ./autogen.sh
    make

NB: I recommend people to use Monodevelop
